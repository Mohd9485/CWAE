# Conditional Wasserstein Autoencoders (CWAE)

This repository is by [Mohammad Al-Jarrah](https://sites.google.com/view/mohammadaljarrah), Michele Martino, Marcus Yim, [Bamdad Hosseini](https://bamdadhosseini.org/), [Amirhossein Taghvaei](https://amirtag.github.io) and contains the Pytorch source code to reproduce the experiments in our 2026 paper [Conditional Sampling via Wasserstein Autoencoders and Triangular Transport](https://arxiv.org/abs/2604.02644). 

## Python libraries needed
* Numpy
* PyTorch
* Scipy
* Seaborn
* [Python Optimal Transport (POT)](https://pythonot.github.io) 

## Flow Field Data
* Generated via [LBM simulation](https://github.com/mohamedayoub97/Lattice-Boltzmann-Simulation-LBS)
